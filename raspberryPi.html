setting up your Raspberry Pi

NOTE: some text here

What you will need:

  • SD Card (8GB) pre-installed with Raspberry Pi NOOBS
  • Display cable – HDMI to DVI cable
  • Standard USB Keyboard and Mouse
  • Power supply

    • - The Pi is powered by a USB Micro power supply that can supply at least 2A at 5V for the Model 3B

  • Ethernet cable

Getting NOOBS and installing it on an SD Card:

NOOBS is available for download HERE on the Raspberry Pi website.

NOTE: be sure to get the Offline and network install ZIP file

Once you’ve downloaded the NOOBS zip file, you will need to copy the contents to a formatted SD card on your computer.

  1. Format and SD card (8GB) as FAT
  2. Extract files and contents from the NOOBS zip file.
  3. Copy extracted file at the root of directory of the SD card. Note: copy files without its folder
  4. Insert SD card into Raspberry Pi
  5. Power up Raspberry Pi
  6. On first boot, the FAT partition will automatically be resized.
  7. Selecting OS: “Raspbian [RECOMMENDED]”
